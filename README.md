# Favotire channels

A CRUD example using MongoDB and NodeJS

![](./readme-src/favorite-channels.png)

## How to run

1. Clone the repo
2. Run `npm i`
3. Create a .env file
4. Add to .env file `PORT` and `MONGO_URI` var
5. Run `npm run start`
