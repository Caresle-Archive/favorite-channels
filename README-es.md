# Favorite channels

Un ejemplo de CRUD usando MongoDB y NodeJS

![](./readme-src/favorite-channels.png)

# Como ejecutar

1. Clona el repositorio
2. Ejecuta `npm i`
3. Crea un archivo `.env`.
4. Agrega al archivo .env las variables `PORT` y `MONGO_URI`
5. Ejecuta `npm run start`
